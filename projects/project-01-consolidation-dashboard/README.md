# ♡ Project 01 — Consolidation Dashboard

## ✦ Overview

This project was developed individually as part of the **Data and Artificial Intelligence Engineering** program at **Faculdade Infnet**.

The objective was to create an interactive financial dashboard using **Google Sheets** as the data source and **Looker Studio** for data visualization, following the requirements provided by the course.

---

## ✦ Objective

Develop an interactive dashboard capable of presenting financial information through key performance indicators and visualizations, applying concepts of data organization, dashboard design and business intelligence.

---

## ✦ Dataset

The dataset was provided as part of the academic activity and imported into **Google Sheets**, which served as the data source for the dashboard.

---

## ✦ Technologies

- Google Sheets
- Looker Studio

---

## ✦ Dashboard Features

The dashboard includes:

- Average Balance KPI
- Maximum Balance KPI
- Minimum Balance KPI
- Balance time series
- Detailed transaction table
- Date range filter
- Last updated information
- Customized layout and branding
