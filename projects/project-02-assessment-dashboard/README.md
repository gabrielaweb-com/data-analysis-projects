# ♡ Project 02 — Academy Attendance Dashboard

## ✦ Overview

This project was developed individually as part of the **Data and Artificial Intelligence Engineering** program at **Faculdade Infnet**.

The objective was to design and develop a multi-page interactive dashboard using **Google Sheets** as the data source and **Looker Studio** for data visualization, following the requirements provided during the course.

---

## ✦ Objective

Develop an interactive report capable of monitoring attendance across gym classes through key performance indicators, charts and interactive filters, providing an organized and intuitive data visualization experience.

---

## ✦ Dataset

The dataset was provided as part of the academic activity in CSV format and imported into **Google Sheets**, which served as the data source for the dashboard.

The data includes information related to:

* Class date
* Gym branch
* Class modality
* Instructor
* Registered participants
* Attendees

---

## ✦ Technologies

* Google Sheets
* Looker Studio

---

## ✦ Dashboard Structure

The report is composed of four pages:

### Page 1 — Data Overview

* Data table
* Class information
* Dataset exploration

### Page 2 — Performance Indicators

* Total registered participants
* Total attendees
* Attendance by class modality

### Page 3 — Interactive Analysis

* Attendance by gym branch
* Attendance by modality
* Attendance over time
* Branch filter
* Modality filter
* Date range control

### Page 4 — Executive Dashboard

* Key performance indicators
* Interactive charts
* Dashboard controls
* Presentation-ready layout
